# SQLite Memory Corruption
This is a remediation script developed to remediate SQLite < 3.50.2 Memory Corruption Plugin 242325 
and SQLite 3.44.0 < 3.49.1 Plugin 240237 vulnerabilities. Tested on a Microsoft Azure VM and scanned
with Tenable Vulnerability Management Authenticated Scan.

---

# Technology Utilized
- Tenable (enterprise vulnerability management platform)
- Azure Virtual Machines (Nessus scan engine + scan targets)
- PowerShell & BASH (remediation scripts)
- ChatGPT
