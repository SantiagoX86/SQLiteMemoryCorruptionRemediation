# SQLite Memory Corruption
This is a remediation script developed to remediate SQLite < 3.50.2 Memory Corruption Plugin 242325 vulnerability.
Tested on a Microsoft Azure VM and scanned with Tenable Vulnerability Management Authenticated Scan.

---

# Technology Utilized
- Tenable (enterprise vulnerability management platform)
- Azure Virtual Machines (Nessus scan engine + scan targets)
- PowerShell & BASH (remediation scripts)
- ChatGPT
